# Mobile Touch 移动端触摸控制

## 能力描述

在移动端触摸设备上提供与键盘一致的游戏操作体验，包含：自动横屏锁定、触摸按钮布局、分辨率适配、帮助面板控制、自动全屏、以及各游戏界面的移动端适配。

## 触发条件

- 设备检测：`'ontouchstart' in window && window.innerWidth < 1024`
- 自动启用触摸控件，无需用户手动设置

## 功能规格

### 横屏锁定
- 触摸设备启动时调用 `screen.orientation.lock('landscape')`（Promise 使用 .catch 降级）
- 不支持 orientation lock 的浏览器静默降级
- Viewport 禁止用户缩放（maximum-scale=1.0, user-scalable=no）

### 自动全屏
- 点击"开始游戏"时调用 `document.documentElement.requestFullscreen()`
- `.catch(() => {})` 静默降级（不支持全屏的浏览器不报错）
- Android 系统推荐使用

### 触摸按钮映射

所有触摸按钮推送的 action 名称必须与键盘输入 `input._actions` 完全一致：

| 按钮标签 | 推送 Action | 等价键盘 |
|---------|-----------|---------|
| 左摇杆 | WASD keys | WASD |
| 跳 | `jump` | Space |
| 挖（长按持续） | `break` + `attack` | 左键 |
| 放 | `place` | 右键 |
| 闪 | `dodge_KeyW/S/A/D` | 双击 WASD |
| 交互 | `interact` | E |
| 技能 | `skillUseExplore` | X |
| 切换 | `skillCycleExplore` | C |
| 武器 | `weaponSwitch` | Q |
| 飞行 | `flight` | Tab |
| 视角 | `toggleCamera` | F / F5 |
| 帮助 | `help` | H |
| ⏸ | `escape` | ESC |

### 按钮布局

```
┌──────────────────────────────────────────┐
│  ┌──┐                                    │
│  │⏸ │ Status bar     ┌────┬────┬────┐  │
│  └──┘                 │交互│技能│切换│  │
│      镜头拖拽区域      ├────┼────┼────┤  │
│                       │武器│飞行│视角│  │
│  ┌─ 摇杆 ─┐          ├────┴────┴────┤  │
│  │        │          │    帮助      │  │
│  │        │          └──────────────┘  │
│  │        │                             │
│  │        │  [1][2][3][4][5][6][7][8]  │
│  └────────┘    [跳] [挖] [放] [闪]     │
└──────────────────────────────────────────┘
```

- **右功能列**（3×2 + 居中）：交互 / 技能 / 切换 / 武器 / 飞行 / 视角 / 帮助
- **底部合并区**（两行）：快捷栏 + 动作按钮
- **左上角**：暂停按钮
- **左下角**：摇杆

### 按钮尺寸

| 元素 | 尺寸 |
|------|------|
| 功能按钮 | `min(36px, 6vw)` |
| 动作按钮（跳/挖/放/闪） | `min(40px, 6vw)` |
| 快捷栏 slot | `min(36px, 5vw) × min(36px, 5vw)` |
| 暂停按钮 | `min(36px, 6vw)` |
| 摇杆底座 | `min(90px, 16vw)` |
| 摇杆 knob | `min(36px, 6vw)` |

### 分辨率适配

- 按钮尺寸基于 `min(固定px, vw)` 计算
- 基准：VIVO X200S 横屏 1260px 宽度
- 小屏降级：375px 宽度下按钮不小于 30px
- 大屏上限：按钮最大 36-40px

### 底部区域

- 快捷栏和动作按钮合并为一个底部面板（bottom:10px）
- 操作栏在快捷栏下方
- 触摸模式下桌面 `#hotbar` 隐藏

### 帮助面板

- 触摸模式下 `hud.setTouchMode(true)` 时：
  - `toggleHelp()` 不做任何事
  - `showHelp()` 不做任何事
  - `H` 触摸按钮在触摸模式下应无响应

### 暂停界面移动端适配

- 手机（<480px）上允许内部滚动
- 角色预览（右侧）隐藏
- 角色卡片从 2 列改为 1 列
- 缩小内边距和字体

### 背包系统移动端适配

- 移除 `min-width: 500px`，使用响应式宽度
- 手机（<480px）上单列垂直堆叠，允许内部滚动
- 存储网格 3 列（格子 32px）
- 触摸设备上拖拽改为点击拾取 + 点击放置
- 打开背包时隐藏触摸控件，关闭时恢复

### 开始界面移动端适配

- 手机（<480px）上允许内部滚动
- 隐藏左侧照片
- 缩小标题字体

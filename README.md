## ✨ Core Features

### 📱 Touch & Mobile
- Complete touch control scheme: virtual joystick movement, camera drag, action buttons
- Auto landscape lock, auto fullscreen request
- Responsive UI, adapts from small phones to large tablets
- Touch-based inventory interaction (tap to pick up + tap to place)

### ⚔️ Combat System
- 5 weapons: Wooden Sword, Stone Sword, Iron Sword, Diamond Sword, Netherite Sword
- 8 hostile mobs: Zombie, Skeleton, Spider, Cave Spider, Creeper, Enderman, Wolf, Slime
- Block (50% damage reduction), Dodge (double-tap direction key)
- Ranged bow system (charge to shoot)
- Floating damage numbers, item drop on death

### 🎵 Music & Sound Effects
- Procedurally generated background music via Web Audio API (multi-layer 32-second loop)
- 15 sound effects: break, place, attack, hurt, pickup, upgrade, explosion, etc.
- Independent volume control for BGM and SFX

### 🔧 Crafting System
- 2×2 crafting grid, drag-and-drop operation
- 20+ crafting recipes: full chain from tools to weapons to blocks
- Complete upgrade path from wood to diamond

### 🗺️ Level System
- 6 selectable levels: Peaceful Plains, Mining Hills, Sandy Desert, Misty Swamp, Ice Tundra, Deep Caverns
- Each level has its own quests and progress tracking
- Level switching preserves inventory and skills

### 🌍 More Features
- 10x large map (1280 chunks), chunk-based loading and LOD
- Day/Night cycle (10-minute full cycle)
- Flight mode (Tab to toggle)
- 4 camera modes (First Person / Third Person Back / Third Person Front / God View)
- 5 selectable characters (Steve, Predator, Twilight Mage, Cave Walker, etc.)
- Pet follow system
- Skill system (exploration skills)
- World save (persisted via localStorage)

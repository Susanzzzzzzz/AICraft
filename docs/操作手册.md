# AICraft 操作手册

## 1. 游戏简介

AICraft 是一款 3D 体素沙盒网页游戏。你可以在方块构建的 3D 世界中自由探索，收集资源、合成工具和武器、与怪物战斗、完成关卡目标。

### 目标
- 探索每个关卡并完成任务
- 收集资源，合成更强大的装备
- 击败敌对生物，生存下来

## 3. 桌面端操控

### 3.1 基本移动

| 操作 | 按键 | 说明 |
|------|------|------|
| 前进 | W | 朝面朝方向移动 |
| 后退 | S | 向后移动 |
| 左移 | A | 向左移动 |
| 右移 | D | 向右移动 |
| 跳跃 | 空格键 | 跳过障碍物 |
| 视角转动 | 鼠标移动 | 环顾四周 |

### 3.2 方块操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 破坏方块 | 鼠标左键 | 按住持续挖掘 |
| 放置方块 | 鼠标右键 | 在目标方块相邻面放置 |
| 选中方块 | 鼠标中键 | 快速选中瞄准的方块 |
| 选择方块 | 数字键 1-8 | 切换快捷栏中的方块 |

### 3.3 战斗操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 攻击 | 鼠标左键 | 攻击准星指向的怪物 |
| 切换武器 | Q | 循环切换已拥有的武器 |
| 格挡 | 按住 G | 减伤 50%，移动速度降低 |
| 闪避 | 双击 WASD | 朝对应方向快速闪避 |
| 蓄力射击 | 按住左键（持弓时） | 蓄力越久伤害越高 |

### 3.4 功能操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 打开背包 | E | 查看物品、合成、装备护甲 |
| 飞行模式 | Tab | 切换飞行/行走 |
| 使用技能 | X | 使用当前选中的探索技能 |
| 切换技能 | C | 切换已解锁的探索技能 |
| 切换视角 | F / F5 | 循环切换 4 种视角模式 |
| 帮助 | H | 显示/隐藏详细操作面板 |
| 暂停 | ESC | 暂停游戏，打开暂停菜单 |
| 重置位置 | R | 回到出生点 |
| 聊天 | T | 输入聊天消息 |
| 命令 | / | 输入斜杠命令（聊天中） |

### 3.5 可用命令

| 命令 | 示例 | 说明 |
|------|------|------|
| `/tp` | `/tp 10 20 30` | 传送至指定坐标 |
| `/give` | `/give diamond 5` | 获取物品 |
| `/time` | `/time 0.5` | 设置时间（0-1） |
| `/fly` | `/fly` | 切换飞行 |
| `/help` | `/help` | 显示命令帮助 |

## 4. 移动端操控

### 4.1 触屏布局

| 控件 | 位置 | 操作方式 |
|------|------|----------|
| 摇杆 | 左下角 | 拖拽控制移动方向 |
| 镜头拖拽 | 右侧区域 | 滑动控制视角 |
| 暂停 | 左上角 | 点击暂停/继续 |
| 操作按钮 | 底部中央 | 跳跃、挖掘（长按持续）、放置、闪避 |
| 方块选择 | 底部快捷栏 | 点击方块预览切换 |
| 功能按钮 | 右上角 | 交互、技能、切换、武器、飞行、视角、帮助 |

### 4.2 功能按钮说明

| 按钮 | 功能 | 对应桌面操作 |
|------|------|-------------|
| 交互 | 打开背包/骑乘 | E 键 |
| 技能 | 使用当前技能 | X 键 |
| 切换 | 切换技能 | C 键 |
| 武器 | 切换武器 | Q 键 |
| 飞行 | 切换飞行模式 | Tab 键 |
| 视角 | 切换视角模式 | F 键 |
| 帮助 | 操作提示（触屏模式下不可用） | H 键 |

### 4.3 移动端特殊操作

- **挖掘**：长按持续挖掘，松开停止
- **闪避**：点击触发全方向闪避
- **方块选择**：点击底部方块预览直接切换
- **自动全屏**：点击"开始游戏"后自动请求全屏
- **横屏锁定**：进入游戏时自动锁定横屏方向

## 5. 系统详解

### 5.1 背包与物品

背包分为三个区域：
- **存储区**（72 格）：存放所有物品，支持翻页（‹ 1/2 ›）
- **快捷栏**（8 格）：快速切换和使用，对应数字键 1-8
- **护甲槽**（4 格）：头盔、胸甲、护腿、靴子

**桌面端操作：**
- 拖拽物品到合成网格
- 点击存储区物品移动到快捷栏
- 右键快捷操作

**移动端操作：**
- 点击物品拾取（高亮），再点击目标槽放置
- 点击同一物品取消选中

### 5.2 合成系统

2×2 合成网格，支持 20+ 配方：

| 产物 | 材料 |
|------|------|
| 木板 x4 | 木头 x1 |
| 木棍 x4 | 木板 x2 |
| 木剑 | 木板 x2 + 木棍 x1 |
| 石剑 | 石头 x2 + 木棍 x1 |
| 铁剑 | 铁锭 x2 + 木棍 x1 |
| 钻石剑 | 钻石 x2 + 木棍 x1 |
| 木镐 | 木板 x3 + 木棍 x2 |
| 石镐 | 石头 x3 + 木棍 x2 |
| 铁镐 | 铁锭 x3 + 木棍 x2 |
| 钻石镐 | 钻石 x3 + 木棍 x2 |
| 各种斧头、铲子 | 对应材料 + 木棍 |

## 6. 小贴士

1. **第一天很重要**：优先收集木头，制作基础工具
2. **善用飞行**：按 Tab 开启飞行模式，方便探索
3. **及时存档**：游戏每 30 秒自动保存一次
4. **优先做护甲**：钻石护甲大幅提升生存能力
5. **切换关卡**：暂停菜单中可选择不同关卡
6. **横屏游玩**：移动端横屏模式提供更大操作空间

# AICraft 操作手册

## 一、游戏简介

AICraft 是一款 3D 体素沙盒游戏网页版。你可以在一个由方块构成的 3D 世界中自由探索、采集资源、合成工具武器、与怪物战斗、完成关卡任务。

### 目标
- 探索各个关卡，完成任务
- 收集资源，合成更强大的装备
- 击败敌对生物，生存下去

## 三、桌面端操作

### 3.1 基础移动

| 操作 | 按键 | 说明 |
|------|------|------|
| 前进 | W | 朝视线方向移动 |
| 后退 | S | 向后移动 |
| 左移 | A | 向左平移 |
| 右移 | D | 向右平移 |
| 跳跃 | 空格 | 越过障碍物 |
| 视角控制 | 鼠标移动 | 环顾四周 |

### 3.2 方块操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 破坏方块 | 鼠标左键 | 长按持续挖掘 |
| 放置方块 | 鼠标右键 | 在目标方块相邻面放置 |
| 拾取方块 | 鼠标中键 | 快速选中目标方块 |
| 选择方块 | 数字键 1-8 | 切换快捷栏中的方块 |

### 3.3 战斗操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 攻击 | 鼠标左键 | 攻击准星指向的生物 |
| 切换武器 | Q | 循环切换已拥有的武器 |
| 格挡 | G 按住 | 减伤 50%，移动速度降低 |
| 闪避 | 双击 WASD | 快速向对应方向闪避 |
| 蓄力射击 | 左键按住（持弓时） | 蓄力越久伤害越高 |

### 3.4 功能操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 打开背包 | E | 查看物品、合成、装备护甲 |
| 飞行模式 | Tab | 切换飞行/步行 |
| 使用技能 | X | 使用当前选中的探索技能 |
| 切换技能 | C | 在已解锁的探索技能间切换 |
| 切换视角 | F / F5 | 循环切换 4 种视角模式 |
| 操作帮助 | H | 显示/隐藏详细操作面板 |
| 暂停 | ESC | 暂停游戏，打开暂停菜单 |
| 重置位置 | R | 回到出生点 |
| 聊天 | T | 输入聊天消息 |
| 命令 | / | 输入斜杠命令（在聊天中） |

### 3.5 可用命令

| 命令 | 示例 | 说明 |
|------|------|------|
| `/tp` | `/tp 10 20 30` | 传送到指定坐标 |
| `/give` | `/give diamond 5` | 获取物品 |
| `/time` | `/time 0.5` | 设置时间（0-1） |
| `/fly` | `/fly` | 切换飞行 |
| `/help` | `/help` | 显示命令帮助 |

## 四、移动端操作

### 4.1 触控布局

| 控件 | 位置 | 操作方式 |
|------|------|----------|
| 摇杆 | 左下角 | 拖动控制移动方向 |
| 镜头拖拽 | 右侧区域 | 滑动控制视角 |
| 暂停 | 左上角 | 点击暂停/恢复 |
| 动作按钮 | 底部中间 | 跳、挖（长按持续）、放、闪 |
| 方块选择 | 底部热键栏 | 点击方块预览切换 |
| 功能按钮 | 右上角 | 交互、技能、切换、武器、飞行、视角、帮助 |

### 4.2 功能按钮说明

| 按钮 | 功能 | 等价桌面操作 |
|------|------|-------------|
| 交互 | 打开背包/骑乘 | E 键 |
| 技能 | 使用当前技能 | X 键 |
| 切换 | 切换技能 | C 键 |
| 武器 | 切换武器 | Q 键 |
| 飞行 | 切换飞行模式 | Tab 键 |
| 视角 | 切换视角模式 | F 键 |
| 帮助 | 操作提示（触摸模式无效） | H 键 |

### 4.3 移动端特殊操作

- **挖**：长按持续挖掘，松手停止
- **闪**：点击触发全方向闪避
- **方块选择**：点击底部的方块预览直接切换
- **自动全屏**：点击"开始游戏"后自动请求全屏
- **横屏锁定**：进入游戏自动锁定横屏

## 五、系统详解

### 5.1 背包与物品

背包分为三部分：
- **存储区**（36 格）：存放所有物品
- **快捷栏**（8 格）：快速切换使用，对应数字键 1-8
- **护甲栏**（4 格）：头盔、胸甲、护腿、靴子

**桌面端操作：**
- 拖放物品到合成格
- 点击存储区物品移至快捷栏
- 右键快捷操作

**移动端操作：**
- 点击物品拾取（高亮），再点击目标格放置
- 点击同一物品取消选择

### 5.2 合成系统

2×2 合成网格，支持 20+ 配方：

| 成品 | 材料 |
|------|------|
| 木板 ×4 | 木头 ×1 |
| 木棍 ×4 | 木板 ×2 |
| 木剑 | 木板 ×2 + 木棍 ×1 |
| 石剑 | 石头 ×2 + 木棍 ×1 |
| 铁剑 | 铁锭 ×2 + 木棍 ×1 |
| 钻石剑 | 钻石 ×2 + 木棍 ×1 |
| 木镐 | 木板 ×3 + 木棍 ×2 |
| 石镐 | 石头 ×3 + 木棍 ×2 |
| 铁镐 | 铁锭 ×3 + 木棍 ×2 |
| 钻石镐 | 钻石 ×3 + 木棍 ×2 |
| 各类斧头、铲子 | 对应材料 + 木棍 |

## 六、小贴士

1. **第一天很重要**：先收集木头，制作工具
2. **善用飞行**：Tab 开启飞行模式，探索更方便
3. **及时存档**：游戏每 30 秒自动存档
4. **护甲优先**：钻石护甲大幅提升生存能力
5. **切换关卡**：可以在暂停菜单中选择其他关卡
6. **手机横屏**：建议横屏游玩，操作空间更大

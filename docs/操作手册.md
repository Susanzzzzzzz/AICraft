# AICraft Operation Manual

## 1. Game Introduction

AICraft is a 3D voxel sandbox web game. You can freely explore a block-built 3D world, gather resources, craft tools and weapons, fight monsters, and complete level objectives.

### Goals
- Explore each level and complete tasks
- Collect resources and craft stronger equipment
- Defeat hostile mobs and survive

## 3. Desktop Controls

### 3.1 Basic Movement

| Action | Key | Description |
|--------|------|-------------|
| Move Forward | W | Move in the direction you are looking |
| Move Backward | S | Move backward |
| Strafe Left | A | Move left |
| Strafe Right | D | Move right |
| Jump | Space | Jump over obstacles |
| Look Around | Mouse Move | Look around the environment |

### 3.2 Block Operations

| Action | Key | Description |
|--------|------|-------------|
| Break Block | Mouse Left Button | Hold to continuously mine |
| Place Block | Mouse Right Button | Place on the adjacent face of a target block |
| Pick Block | Mouse Middle Button | Quickly select the targeted block |
| Select Block | Number Keys 1-8 | Switch blocks in the hotbar |

### 3.3 Combat Controls

| Action | Key | Description |
|--------|------|-------------|
| Attack | Mouse Left Button | Attack the mob at the crosshair |
| Switch Weapon | Q | Cycle through owned weapons |
| Block | Hold G | Reduce damage by 50%, movement speed reduced |
| Dodge | Double-tap WASD | Quickly dodge in the corresponding direction |
| Charged Shot | Hold Left Click (when holding Bow) | Longer charge = higher damage |

### 3.4 Function Controls

| Action | Key | Description |
|--------|------|-------------|
| Open Inventory | E | View items, craft, equip armor |
| Flight Mode | Tab | Toggle flying / walking |
| Use Skill | X | Use the currently selected exploration skill |
| Switch Skill | C | Cycle through unlocked exploration skills |
| Switch View | F / F5 | Cycle through 4 camera modes |
| Help | H | Show / hide detailed control panel |
| Pause | ESC | Pause the game, open pause menu |
| Reset Position | R | Return to spawn point |
| Chat | T | Enter chat message |
| Command | / | Enter slash command (in chat) |

### 3.5 Available Commands

| Command | Example | Description |
|---------|---------|-------------|
| `/tp` | `/tp 10 20 30` | Teleport to specified coordinates |
| `/give` | `/give diamond 5` | Obtain items |
| `/time` | `/time 0.5` | Set time (0-1) |
| `/fly` | `/fly` | Toggle flight |
| `/help` | `/help` | Show command help |

## 4. Mobile Controls

### 4.1 Touch Layout

| Control | Position | Operation |
|---------|----------|-----------|
| Joystick | Bottom left | Drag to control movement direction |
| Camera Drag | Right area | Swipe to control camera |
| Pause | Top left | Tap to pause / resume |
| Action Buttons | Bottom center | Jump, Break (hold to continue), Place, Dodge |
| Block Selection | Bottom hotbar | Tap block preview to switch |
| Function Buttons | Top right | Interact, Skill, Switch, Weapon, Fly, View, Help |

### 4.2 Function Button Descriptions

| Button | Function | Equivalent Desktop Action |
|--------|----------|---------------------------|
| Interact | Open inventory / mount | E key |
| Skill | Use current skill | X key |
| Switch | Switch skill | C key |
| Weapon | Switch weapon | Q key |
| Fly | Toggle flight mode | Tab key |
| View | Switch camera mode | F key |
| Help | Operation tips (inactive in touch mode) | H key |

### 4.3 Mobile-Specific Controls

- **Break**: Hold to continuously mine, release to stop
- **Dodge**: Tap to trigger omnidirectional dodge
- **Block Selection**: Tap the block preview at the bottom to switch directly
- **Auto Fullscreen**: Automatically requests fullscreen after clicking "Start Game"
- **Landscape Lock**: Automatically locks to landscape orientation when entering the game

## 5. System Details

### 5.1 Inventory & Items

The inventory is divided into three sections:
- **Storage** (36 slots): Store all items
- **Hotbar** (8 slots): Quick switching and use, corresponding to number keys 1-8
- **Armor Slots** (4 slots): Helmet, Chestplate, Leggings, Boots

**Desktop Controls:**
- Drag and drop items into the crafting grid
- Click items in Storage to move to Hotbar
- Right-click for quick actions

**Mobile Controls:**
- Tap an item to pick it up (highlighted), then tap a target slot to place it
- Tap the same item to deselect

### 5.2 Crafting System

2x2 crafting grid, supports 20+ recipes:

| Result | Materials |
|--------|-----------|
| Wood Planks x4 | Wood x1 |
| Stick x4 | Wood Planks x2 |
| Wooden Sword | Wood Planks x2 + Stick x1 |
| Stone Sword | Stone x2 + Stick x1 |
| Iron Sword | Iron Ingot x2 + Stick x1 |
| Diamond Sword | Diamond x2 + Stick x1 |
| Wooden Pickaxe | Wood Planks x3 + Stick x2 |
| Stone Pickaxe | Stone x3 + Stick x2 |
| Iron Pickaxe | Iron Ingot x3 + Stick x2 |
| Diamond Pickaxe | Diamond x3 + Stick x2 |
| Various axes, shovels | Corresponding material + Stick |

## 6. Tips

1. **The first day matters**: Collect wood first, craft basic tools
2. **Use flight wisely**: Press Tab to enable flight mode for easier exploration
3. **Save often**: The game auto-saves every 30 seconds
4. **Prioritize armor**: Diamond armor greatly improves survivability
5. **Switch levels**: You can select different levels from the pause menu
6. **Play in landscape**: Landscape mode on mobile provides more room for controls
